# Netflix Content Analysis

## Project Overview

This project analyzes Netflix's content library using SQL and Google Sheets.
The objective was to identify trends in content distribution, release history,
ratings, countries, directors, and metadata quality.

---

## Business Problem

Netflix leadership wants an overview of its content library to better
understand production trends and support future licensing and investment
decisions.

---

## Dataset

Netflix Titles Dataset

8,807 titles

Includes Movies and TV Shows

## Key Findings

- Movies account for approximately 70% of the catalog.
- The United States contributes the most content.
- TV-MA is the most common content rating.
- Most titles were released after 2015.
- Director metadata contains over 2,600 missing values.

---

## Recommendations

- Improve metadata quality.
- Continue investment in international content.
- Monitor rating distribution.
- Track long-term release trends.
